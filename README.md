# Sunil's AI Guides

Practical, source-verified guides for building and operating AI agent systems.

These guides come from systems I have built and operated myself. Each public guide is sanitized before publication: no private prompts, credentials, personal data, company information, or internal research notes.

## Standards

- Personal operating evidence comes before generic advice.
- Version-sensitive instructions include a verification date.
- Fast-changing setup details link to current official documentation.
- Cloud and local model boundaries are stated explicitly.
- Tokens, API keys, private configurations, and personal data are never published.

The private research and editorial system behind these guides is not included in this repository.
